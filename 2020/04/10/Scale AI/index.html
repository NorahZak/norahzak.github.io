Scale AI

1.简介

Scale AI 可以帮助用户获取机器学习所需的训练数据集，用户可以提交数据标注任务，并获得返回的数据标注。它使用机器学习模型对数据进行预标注，然后再人工修正。支持多种数据标注任务（2d目标检测框，语义分割等），并提供了多种语言(Python, JavaScript, Ruby)的API供用户调用。

Scale AI 的API分为两种：

  • On-demand API：按需模式，支持的数据标注任务种类有限
  • Enterprise API：企业模式，支持所有种类的数据标注任务，而且能够保证数据标注的准确率

2.支持的数据标注任务

计算机视觉

  • 2D Box Annotation
  • polygon annotation
  • segment annotation
  • video box annotation
  • cuboid annotation
  • Image Categorization

自然语言处理

  • text classification
  • named entity recognition
