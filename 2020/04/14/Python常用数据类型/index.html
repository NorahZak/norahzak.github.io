Python常用数据类型

2020-04-14

#Python

数字(number)

  • 数字是不可变对象，即它的值是不能修改的。
  • 包括整型int，浮点型float等等。它们的最常用的方法就是各种代数运算。
  • 可以用科学计数法来创建数字对象，此时默认的数据类型为浮点数float. 比如a=1e2 -> a=100.0

字符串(string)

  • 与C/C++不同，在Python中没有字符类型，只有字符串类型。字符串是一种不可变对象。

  • operator'[start:end:step]' -> str : Return a subsequence. Be aware that end>=0. For example, s=abc, then s[2:-1:-1] results in cb, not cba. If you want to get cba, use s[2::-1] instead

  • replace(oldstr, newstr[, max]) -> str：将所有oldstr的出现替换为newstr，返回替换后的新子串。如果指定了max参数，则最大替换次数不超过max

  • split(str=’ ‘) -> list：将字符串用分隔符str分割成若干子串，返回子串组成的列表

  • re.split(str, srcstr)：同样是分割字符串，但re.split支持多个分隔符，只需在str中用’|’隔开即可，而Python内置函数split仅支持一个分隔符

  • startswith(str) -> bool : Return True if string starts with str else False.

  • join(seq) -> str : Concatenate strings in seq using self. e.g. '-'.join('a', 'b', 'c') = 'a-b-c'

  • 格式化：Python支持三种字符串格式化方法。

    1. %操作符：用%引导的格式化字符表示字符串中待替换的部分，用%引导的单个变量或者元组来依次给每个格式化字符赋值，格式为'%[flag][width].[precision]typecode' % var。格式化字符的各部分作用如下：

    • width表示显示宽度，flag表示当实际值的宽度不足width时的行为，flag=None表示在前面补空格，flag=0表示在前面补0，flag=-表示在后面补空格。

    • precision表示小数点后精度

    • 示例：print('%04d, %.3f' % (23, 0.56923)) >>> 0023, 0.569

    1. f-string：以 f 或 F 修饰符引领的字符串（f'xxx' 或 F'xxx'），以大括号 {} 标明被替换的字段
    2. format()

  • 正则表达式

    A regular expression (or RE) specifies a set of strings that matches it. The characters in RE can be divided into 2 categories: normal characters and special characters. Some of the special characters are :

    • '.': matches any character except a newline.

    • '^': matches the start of the string.

    • '*': match 0 or more repetitions of the preceding RE, as many repetitions as are possible.

    • '+': match 1 or more repetitions of the preceding RE.

    • '?': match 0 or 1 repetitions of the preceding RE

    • '\': escapes special characters

    • '[]': indicate a set of characters.

    • '|': A|B, where A and B can be arbitrary REs, creates a regular expression that matches either A or B

    • Python support for regular expression operations : module re

元组(tuple)

  • 线性数组。元组是不可变对象。
  • 创建元组：t = () or t = (e1, e2) or t = tuple (object)
  • [i]: 返回下标为i的元素

列表(list)

  • 与元组的区别在于，列表是可变对象，元组是不可变对象。

  • 创建列表：l = [] or l = [e1, e2, ...] or l = list (object)

  • append (e) : 向列表末尾插入元素e.

  • pop(index=-1) -> object : 删除列表的第index个元素并将其返回

  • >/</== : 比较两个列表的大小，方法是从第一个元素开始，依次比较相同位置上的元素大小，直到两个元素不同或者其中一个列表为空（此时非空的那个列表更大）

  • in / not in：用来判断一个元素是否存在于列表中

  • index(element) -> int : 用来查找特定元素在列表中第一次出现的位置，若不存在会报错

  • sort(key=None, reverse=False) -> None : 对列表进行原地排序

    • key : A function that map an element to its key. The sorting is based on the value of key.
    • reverse : True when sort in descending order.

  • [i:j] -> list :返回子列表[i, j-1]。其中i和j可以是负值，表示元素的倒序。

  • * : 将一个列表复制若干次来获得一个新链表。但要注意，*运算符进行的是浅复制，也就是只复制元素的引用。这意味着，如果一个元素是可变对象（比如列表），那么对该元素的其中一个引用进行操作，会修改该对象的值，从而影响其他的引用。

字典(dict)

  • 一个能够存储键值对的无序集合

  • 创建：d = {}

  • 添加元素：d[key]=value

  • keys()：Returns a dict_keys object consists of all keys in dict.

  • values(): Returns a dict_values object (a kind of list) consists of all values in dict.

  • get(key, default=None): If key exists, then return a reference to dict[key], otherwise *return** *default.

  • setdefault(key, default=None): If key exists, then return a reference to dict[key], otherwise *add a new element** (*key, default) to dict and return a reference to default.

  • pop(key) : Remove an element.

  • update(new_dict): Update dict with new_dict. For each (k, v) in new_dict, if k is not in dict, then add (k, v) to dict; else change the value of k to v.

  • items() : Return a list of tuples representing each (k, v) pair in dict.

  • 字典中的关键字必须是不可变对象。

集合(set)

  • 用于存储元素的无序集合
  • 创建集合: s=set() s=set(e1, e2, ...) s=set (object)
  • add(value)
  • remove(value)
  • operator: in

除了Python内置的常用数据类型之外，有许多Python第三方库提供了其他有用的数据类型。

默认字典(defaultdict)

  • defaultdict类是Python内置字典dict类的子类，定义在collections模块中。它们的基本用法相似，不同的是，defaultdict类可以更方便地在访问不存在的元素时插入新元素并进行初始化。
  • defaultdict(default_factory): defaultdict类的构造函数，default_factory是默认的mapped value的类型，比如：default_factory=int意味着新元素会自动被初始化为0，default_factory=list意味着新元素会自动被初始化为空列表。

优先队列(PriorityQueue)

  • 定义在queue模块中。推荐使用heapq模块，比PriorityQueue更有效率。

  • Attributes:

    • queue: A list consisting all elements in the queue.

    • put(e)

    • get()

命名元组(namedtuple)

  • 用于保存结构化数据，相当于C/C++中的结构体，但是不可修改。定义在collections中。
