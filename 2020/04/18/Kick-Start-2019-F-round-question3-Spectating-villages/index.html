Kick Start 2019 F round question3: Spectating villages

2020-04-18

#kickstart #programming

问题描述

The countryside of Kickstartia consists of V villages (labelled from 1 to V), connected by V-1 bidirectional roads (labelled from 1 to V-1). The i-th road connects village Xi to village Yi. Each road connects exactly two villages, and no two roads connect the same two villages. Furthermore, there is exactly one sequence of roads that connects any two villages in Kickstartia.

Some villages are more beautiful than others. The i-th village has a beauty value of Bi. Note that it is possible for a village to have a negative beauty value!

You are going to build lighthouses in some of the villages. A village is illuminated if there is a lighthouse built in it, or there is a lighthouse built in a village that is directly connected to it by a road.

You may build as many or as few (even zero) lighthouses as you like. What is the maximum possible sum of beauty values of illuminated villages you can obtain?

解决方案

所有城市和它们之间的通路共同构成一张图，城市是图的节点，通路则是图的边。由于图是全连通的，而且只有V-1条边，因此这个图其实是一颗树。我们可以选择任意一个节点作为树的根节点，比如1号节点。

定义f(k, p, q)为以k号节点为根的子树的最大beauty value，p和q都是布尔值，分别表示k号节点是否放置了灯塔以及k的父节点是否放置了灯塔。这样，max(f(1, 1, 0), f(1, 0, 0))就是所求答案。

这个函数可以递归地求解，递归的情况分为以下三种：

  1. p=0,q=1：此时，k被它的父节点照亮，而k自己没有灯塔，因此不会影响它的子节点。所以此时f(k, 0, 1) = Bk + sum(max(f(c, 1, 0), f(c, 0, 0))) for all k's child c

  2. p=1：此时f(k, 1, q) = Bk + sum(max(f(c, 1, 1),f(c, 0, 1))) for all k's child c

  3. p=0,q=0：这种情况比较复杂，不仅要考虑到每颗子树的最大beauty value，还要考虑k是否会被它的某一个子节点照亮。我们想要知道，当k的所有孩子都没有灯塔，以及当k的至少一个孩子节点有灯塔，这两种情况下子树的beauty value之和的最大值。可以用动态规划的方法求解：设dp(i, j)表示当k的前i个孩子都没有灯塔(j=0)，以及当k的前i个节点中至少一个有灯塔(j=1)时，前i棵子树的beauty value之和的最大值。则递推关系如下，其中c[i]是k的第i个子节点：

    • dp(i, 0) = dp(i-1, 0)+f(c[i], 0, 0),
    • dp(i, 1) = max(dp(i-1, 1)+f(c[i], 0, 0), max(dp(i-1, 0), dp(i-1, 1))+f(c[i], 1, 0))

    则最终f(k, 0, 0)=max(dp(m, 0), dp(m, 1)+Bk)，其中m是k的子节点数目.

复杂度分析：如果在递归过程中，将递归的解记录下来，那么每个节点只会被求解一次。在对每个节点k求解时，需花费O(m)的时间，其中m是k的子节点数目。所有节点的子节点数目之和为V-1，因此总的时间复杂度为O(V).
