Kick Start 2019 G round question2: The equation

2020-04-15

#kickstart #programming

问题描述

The laws of the universe can be represented by an array of N non-negative integers. The i-th of these integers is Ai.

The universe is good if there is a non-negative integer k such that the following equation is satisfied: (A1 xor k) + (A2 xor k) + … (A\N** xor k) ≤ **M, where xor denotes the bitwise exclusive or.

What is the largest value of k for which the universe is good?

解题思路

由于Ai<=10^15,M<=10^15，因此一定有k<2^50，否则k的最高位超过了Ai和M的最高位，异或之后的和一定是大于M的。对k的第i位，它对总和的贡献与其他位无关，因此我们可以分别考虑每一位对总和的贡献。

假设有zeros[i]个Aj，它们在第i位上是0；有ones[i]个Aj，它们在第i位上是0。那么，k的第i位对总和的贡献就可以求出来了：当它是1时，这个贡献是2^i*zeros[i]，当它是0时，这个贡献是2^i*ones[i]。时间复杂度：O(N*log(max(A)))

接下来，为了使k最大化，我们可以从k的最高位开始，先看能不能将这一位设置为1。如果设置为1，那么它的贡献是2^i*zeros[i]。而对于剩下的位（从i-1到1），我们可以求出它们的和的最小值f(i-1)，只需对一切1<=j<i，比较zeros[j]与ones[j]的大小，就可以确定第j位。那么，如果2^i*zeros[i]+f(i-1)>M，我们就必须将第i位设置为0，否则就将第i位设为1，然后令M=M-2^i*zeros[i]，再重复上述过程决定第i-1位……直到所有位都被确定。时间复杂度：O(N)

总的时间复杂度：O(N*log(max(A)))
